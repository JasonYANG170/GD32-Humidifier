## 功能
- ✅支持4个加湿片同时工作
- ✅支持环境温湿度检测
- ✅支持环境温湿度自适应控制
- ✅支持手动调节温湿度
- ✅支持屏幕显示

如遇问题，请向我提出issues


## 项目参数

* 本设计采用GD32E230C8T6主控，以实现较低成本；
* 本设计采用AHT10，以实现环境监测；
* 本设计采用SSH1106 OLED，以实现数据显示；
